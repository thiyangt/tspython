7.1 Vector files

  1. Shapefiles

  • .shp file contains shape geometry

  • .dbf file holds attributes for each geometry

  • .shx file or shape index file helps link the attributes to the shapes

  1. GeoJSON: Unlike shapefiles, GeoJSON is a single file
